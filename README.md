# Event Handler Platform 🎯

B2B SaaS платформа для организации мероприятий, встреч и проведения опросов. 
Проект представляет собой полнофункциональное веб-приложение с раздельной архитектурой (микросервисный бэкенд и SPA-фронтенд).

## 🚀 Стек технологий

### Бэкенд:
* **Фреймворк:** FastAPI (Python 3.10+)
* **База данных:** PostgreSQL + SQLAlchemy (Asyncio)
* **Миграции:** Alembic
* **Кэширование/Сессии:** Redis
* **Инфраструктура:** Docker & Docker Compose

### Фронтенд:
* **Фреймворк:** React.js
* **Стилизация:** Tailwind CSS
* **Взаимодействие с API:** Axios

## 📂 Структура проекта

* `backend/` — Асинхронное API на фреймворке FastAPI
  * `app/api/` — Роутеры и эндпоинты
  * `app/models/` — ORM-модели базы данных (SQLAlchemy)
  * `app/schemas/` — Валидация данных (Pydantic)
* `frontend/` — Клиентское одностраничное приложение (SPA) на React
  * `src/components/` — Переиспользуемые UI-компоненты
  * `src/pages/` — Страницы интерфейса (авторизация, дашборд, создание ивента)
