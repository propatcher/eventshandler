# Веб-система для управления мероприятиями 🎓

Данный репозиторий содержит исходный код проекта, разрабатываемого в рамках дипломной работы (выпускной квалификационной работы).
Проект представляет собой полнофункциональное веб-приложение с раздельной клиент-серверной архитектурой, предоставляющее RESTful API для организации мероприятий и проведения опросов.

## 🚀 Стек технологий

### Серверная часть (Backend):
* **Фреймворк:** FastAPI (Python 3.10+)
* **База данных:** PostgreSQL + SQLAlchemy (Асинхронный движок)
* **Миграции:** Alembic
* **Кэширование и сессии:** Redis
* **Контейнеризация:** Docker & Docker Compose

### Клиентская часть (Frontend):
* **Библиотека:** React.js
* **Стилизация:** Tailwind CSS
* **Взаимодействие с API:** Axios

## 📂 Структура проекта

* `backend/` — Асинхронное REST API
  * `app/api/` — Роутеры и контроллеры (endpoints)
  * `app/models/` — ORM-модели базы данных
  * `app/schemas/` — Схемы валидации данных (Pydantic)
* `frontend/` — Клиентское одностраничное приложение (SPA)
  * `src/components/` — UI-компоненты
  * `src/pages/` — Основные экраны приложения
