# Event Handler 🎯

B2B SaaS платформа для организации мероприятий, встреч и проведения опросов. 
Разрабатывается в рамках дипломного проекта по созданию высоконагруженного веб-приложения.

## 🚀 Стек технологий

* **Фреймворк:** FastAPI (Python 3.10+)
* **База данных:** PostgreSQL + SQLAlchemy (Asyncio)
* **Миграции:** Alembic
* **Кэширование/Сессии:** Redis
* **Инфраструктура:** Docker & Docker Compose
